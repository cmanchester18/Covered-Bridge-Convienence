⚙️ Firebase Configuration

Enter your Firebase project credentials below. You can find these in your Firebase console under Project Settings → Your Apps → Firebase SDK snippet.

How to get your config
  1. Go to console.firebase.google.com
  2. Create or select a project
  3. Add a Web App to the project
  4. Copy the firebaseConfig object values above
  5. In Firestore, create collections: menu, specials, gallery, hours, contact
  6. Set Firestore Rules to allow read/write (or set up auth)
